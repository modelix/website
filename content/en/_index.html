{{< /blocks/cover >}} {{% blocks/lead color="primary" %}} Modelix is a open source platform that **aims to bring modeling on the web**, which aims for a productive integration between modern web technologies and best-practices from language engineering. We provide collaborative editing of domain-specific models in the browser and allow to interact with models and services around them over web-standard APIs. Right now Modelix can act as an extension or drop in replacement to create webbased editor for existing languages in JetBrains MPS. {{% /blocks/lead %}} {{< blocks/section color="dark" type="features">}} {{% blocks/feature icon="fa-lightbulb" title="See Modelix in action!" url="https://docs.modelix.org/modelix/latest/samples/reference/architecture.html" %}} Get started with our example projects and try modelix yourself. {{% /blocks/feature %}} {{% blocks/feature icon="fab fa-github" title="Contributions welcome!" url="https://github.com/modelix/" %}} We do a [Pull Request](https://github.com/modelix/modelix.core/pulls) contributions workflow on **GitHub**.
